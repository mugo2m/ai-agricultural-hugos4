## 📋 Overview

AI Agricultural Knowledge Base is a sophisticated voice-powered platform that transforms how farmers access agricultural information. Built on Next.js, Firebase, and Google's Gemini AI, it provides:

- 🎤 **Voice-First Interface** - Farmers speak questions naturally
- 🌱 **Personalized Recommendations** - AI-generated advice based on farm profile
- 📊 **Multimodal Responses** - Text + relevant agricultural images
- 🔐 **Secure Authentication** - Phone/email sign-in with Firebase
- 💳 **MPESA Integration** - Secure payments (simulated for testing)

## 🎯 Core Features

| Feature | Description |
|---------|-------------|
| **Voice Q&A** | Ask questions about crops, pests, fertilizers via voice |
| **Farm Profiling** | Collects crop types, location, acreage, livestock data |
| **AI Recommendations** | Gemini generates 5 personalized farming tips |
| **Streaming Voice** | Recommendations read aloud with natural pauses |
| **Conversation History** | All Q&A saved for future reference |
| **Mobile Responsive** | Works on smartphones (crucial for farmers) |
